# 刮刮乐 (Scratch Card Game)

SDL2 图形化刮刮乐模拟游戏，1280×720 固定窗口。

## 快速开始

### 依赖

- SDL2
- SDL2_ttf
- g++ (C++17)

MSYS2 安装：

```bash
pacman -S mingw-w64-x86_64-SDL2 mingw-w64-x86_64-SDL2_ttf
```

### 编译

```bash
g++ -std=c++17 main.cpp game.cpp render.cpp save.cpp \
  $(pkg-config --cflags --libs sdl2 SDL2_ttf) \
  -o guaguale_game
```

### 运行

```bash
./guaguale_game
```

## 玩法

### 选购

主界面展示三种刮刮乐，点击卡片选中（橙色边框），再点击右下角"购买"按钮购买：

| 卡片 | 售价 | 格子数 | 最高奖金 | 期望回报 |
|------|------|--------|----------|----------|
| 小试一手 | 1万 方斯 | 10 格 | 25万 | ~90% |
| 人之常情 | 2万 方斯 | 15 格 | 50万 | ~90% |
| 人，稿子，钻石 | 5万 方斯 | 20 格 | 100万 | ~90% |

### 刮奖

- 鼠标按住拖动涂抹刮开涂层
- 每格上方显示中奖/未中奖图案，下方显示金额
- 右下角"一键全部刮开"跳过手动涂抹
- 刮开 ≥95% 自动弹出结算弹窗（含纪念品 + 中奖金额）

### 作弊按钮

左上角"开滴滴"按钮，每次点击获得 10 万方斯（不计入盈亏统计）：

| 点击次数 | 剧情 | 收入 |
|----------|------|------|
| 普通 | 开了一天滴滴 | 10万 |
| 第 10 次 | 管理局给娜娜莉的医疗补贴下来了 | 100万 |
| 第 20 次 | 背着店长偷偷把伊波恩抵押了… | 1000万 |

剧情每局仅触发一次，重启重置。

### 存档

- **F5** — 手动保存
- **F9** — 手动读取
- 退出时自动保存
- 启动时自动加载
- 存档文件：`guaguale_save.dat`（XOR 加密 + CRC 校验）

## 概率系统

奖金权重采用指数衰减：**weight = exp(-1.2 × prize_wan)**

中奖时各档奖金概率（理论值）：

```
1万  ████████████████████████████████  69.9%
2万  ██████████  21.0%
3万  ███  6.3%
4万  █  1.9%
5万  <1%
…    …
100万  <0.001%（仅5万面额卡片）
```

每张彩票的期望奖金 = 售价 × 90%（略有浮动），即长期期望净亏损 10%。

## 代码结构

```
local/
├── config.h      配置常量、颜色、卡片定义、概率算法
├── save.h/.cpp   加密存档（XOR + CRC32, 二进制格式）
├── render.h/.cpp SDL2 渲染（UI、涂层刮开、弹窗）
├── game.h/.cpp   状态机、输入处理、游戏流程
└── main.cpp      入口
```

分层依赖：`main → Game → Renderer / SaveManager → Config`

## 自定义

- 替换占位图片：修改 `game.cpp` 中 `m_coinTex` 和 `m_cardTex[3]` 的纹理加载
- 调整刮开阈值：修改 `config.h` 中的 `SCRATCH_REVEAL_THRESHOLD`
- 调整笔刷大小：修改 `game.cpp` 中 `scratchAt` 调用的最后一个参数
- 调整期望回报率：修改 `config.h` 中 `calcWinProb` 的系数（当前为 0.9）
